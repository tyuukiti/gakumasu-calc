# 学マス 育成ステータス理論値計算ツール

学園アイドルマスター（学マス）のサポートカード編成によるステータス理論値を計算するデスクトップツール＋Web版です。

## 利用方法

| 種別 | リンク |
|---|---|
| 🌐 Web版 | [tyuukiti.github.io/gakumasu-calc](https://tyuukiti.github.io/gakumasu-calc/) |
| 📦 デスクトップ版 | [Releases](https://github.com/tyuukiti/gakumasu-calc/releases) |

## ⚠️ 注意事項

- **キャラ固有ボーナスと持ち込みPアイテムによるステータス補正は含まれていません。** 表示される数値はサポートカード編成による理論値のみです。実際の育成結果はこれらの要素により変動します。
- プロデュースアイテム効果によるステータス上昇は計算に含まれます（マウスオーバーで `[アイテム]` として内訳表示）。
- 所持カードモードでは凸数に応じた効果値で計算します。全カードモードでは4凸（完凸）前提で計算します。
- ステータス上限は育成プランごとに異なります（初Legend: 2800、NIAマスター: 2600）。

- モダンブラウザ（Chrome, Firefox, Safari, Edge）

cd web && npm install && npm run build
```

## 使い方

### 計算ツール（Web版 / デスクトップ版共通）

1. **育成プランを選択** — 「初Legend」「NIA」などから育成プランを選びます
2. **育成タイプを選択** — センス / ロジック / アノマリーを選びます
3. **属性設定を行う**
   - 各属性（Vo / Da / Vi）のメイン/サブを設定
   - SP枚数（属性ごとのSPカード枚数）を指定
4. **レッスン内イベント回数を設定**
   - テンプレートから選択すると推奨値が自動入力されます
   - 「詳細を編集」を展開すれば個別に調整可能です
5. **「計算実行」をクリック** — 最適な編成パターンが複数表示されます
6. **編成パターンを比較**
   - 各パターンには使用カードのレアリティ・属性・プランとステータス寄与値が表示されます
   - カード名にマウスオーバーでステータス上昇の内訳を確認できます
   - 所持カードモードでは、レンタル枠と所持枠の入れ替え検証も自動で行われます
7. **コンテストモード** — スキルカード・コンテストアイテム付きサポカを除外して計算できます

### CardInventoryManager（所持管理）

1. カード画像をクリックして所持/未所持を切り替えます
2. 凸数をドロップダウンで選択します（0〜4）
3. フィルター（レアリティ/属性/所持状態）で絞り込みできます
4. **Ctrl+S** または「保存」ボタンで所持情報を保存します
5. 保存した所持情報は計算ツールの「所持カードのみで計算」オプションで利用されます

## データ構成

```
Data/
├── SupportCards/     # サポートカードデータ (SSR/SR/R)
├── Plans/            # 育成プランデータ
├── Templates/        # イベント回数テンプレート
├── Images/           # カード画像 + マッピングファイル
└── Inventory/        # 所持情報 (自動生成)
```

# 確認のみ（書き込みなし）
python scripts/sync_wiki.py --dry-run
```
